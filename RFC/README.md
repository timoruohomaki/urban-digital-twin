# Urban Digital Twin Semantic Framework

This repository contains the draft RFC series for the Urban Digital Twin Semantic Framework.

## Purpose

The Semantic Framework defines concepts, terminology and semantic relationships that support interoperability between independently governed semantic domains. It is intentionally implementation-independent.

## Repository structure

- `rfcs/` Core RFC documents
- `mappings/` Non-normative mappings to existing standards
- `diagrams/` Figures used by RFCs
- `examples/` Worked examples
